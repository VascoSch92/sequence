===========
Sequentium
===========

Sequentium is a user-friendly package that implements various well-known sequences,
providing a seamless and intuitive experience for the user.

The package is based on the following three principles:

- Simplicity
- Correctness
- Intuitiveness


If you would like to contribute to the package with your preferred sequence
or suggest something else, please refer to the wiki's `How to Contribute` section.
